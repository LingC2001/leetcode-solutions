## Problem Statement

Given a string `s` that consists of lowercase English letters, return the string obtained by removing all trailing vowels from `s`.

The vowels are `'a'`, `'e'`, `'i'`, `'o'`, and `'u'`.

[View on LeetCode →](https://leetcode.com/problems/trim-trailing-vowels/)

---

## Examples

### Example 1

```
Input: s = "idea"
Output: "id"
Explanation: Removing the trailing "ea" from "idea" gives "id".
```

### Example 2

```
Input: s = "day"
Output: "day"
Explanation: There are no trailing vowels in the string "day".
```

### Example 3

```
Input: s = "aeiou"
Output: ""
Explanation: All characters are vowels, so the result is an empty string.
```

---

## Approach

Use a right pointer starting at the last character:

1. Walk the pointer left as long as the current character is a vowel.
2. Once a non-vowel (consonant) is found, the result is the slice `s[:r+1]`.
3. If the pointer goes past the beginning without finding a consonant, return `""`.

---

## Complexity Analysis

**Time Complexity:** O(n), where n is the length of `s`.

**Space Complexity:** O(1) extra space (excluding the output string).

## Key Insights

- Only the **trailing** characters need to be checked — a single backward scan suffices.
- Using a vowel set (hash map or fixed lookup) makes the membership check O(1).
- No need to rebuild the string; a simple slice of the original is enough.
